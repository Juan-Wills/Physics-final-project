Cinematica

Se llama cinética o cinematica a la especialidad de la física centrada en el análisis del movimiento, dejando de lado el estudio de las fuerzas que lo originan. La cinética, en este marco, suele estudiar la trayectoria de un cuerpo en movimiento en función del tiempo.

MRU

Movimeinto rectilineo uniforme

Se define como el desplazamiento de un objeto en una dirección horizontal con una velocidad y tiempo constante en una determinada distancia.

Sus magnitudes son las siguientes:

    Velocidadspeed
  • velocidad inicial = Vi (m/s)
  • velocidad final = Vf (m/s)
  • velocidad promedio = Vprom => Vx (m/s)
    • Distanciadistance
  • distancia inicial = Xi (m)
  • distancia final = Xf (m)
  • distancia promedio= Xprom => X (m)
    • Tiemposchedule
  • tiempo = t (seg)

Ejercicios
    velocidad promedio
  • Vprom = ΔX/Δt
    • distancia final
  • Xf = Xi+Vprom.t

MRUA

Sus magnitudes son las siguientes:

    Aceleraciónbolt
  • aceleracion = a (m/s2)
    • Velocidadspeed
  • velocidad inicial = Vi (m/s)
  • velocidad final = Vf (m/s)
  • velocidad promedio = Vprom => Vx (m/s)
    • Distanciadistance
  • distancia inicial = Xi (m)
  • distancia final = Xf (m)
  • distancia promedio= Xprom => X (m)
    • Tiemposchedule
  • tiempo = t
Ejercicios
    aceleracion
  • a = ΔVprom/Δt
    • velocidad final
  • Vf = Vi+a.t
  • Vf2 = Vi2+2a.(Xf-Xi)
    • distancia final
  • Xf = Xi+1/2(Vi-Vf).t
  • Xf = Xi+Vi.t+1/2a.t2

CA

Caida Libre

Se define como el movimiento de un objeto en una dirección vertical bajo la acción exclusiva de la gravedad.

Sus magnitudes son las siguientes:

    Aceleraciónbolt

    • Velocidadspeed
  • velocidad inicial = Vi (m/s)
  • velocidad final = Vf (m/s)
  • velocidad promedio = Vprom => Vx (m/s)
    • Distanciadistance
  • distancia inicial = Yi (m)
  • distancia final = Yf (m)
  • distancia promedio= Yprom => Y (m)
    • Tiemposchedule

Ejercicios

Sistema vectorial

Hay muchas magnitudes físicas que tienen intensidad (o módulo) y dirección, como por ejemplo la velocidad o la fuerza, los vectores se usan para representar las magnitudes de este tipo. Estas magnitudes pueden ser representadas de forma grafica por medio de flechas en un espacio cardinal.

Vectores

Introduccion a los planos cartesianos

Un vector es una entidad en física que se caracteriza por tener dos propiedades esenciales: magnitud y dirección. Esto significa que, además de tener un tamaño (magnitud), un vector también señala hacia un determinado rumbo (dirección).

Sus magnitudes son las siguientes:

    Velocidadspeed
  • velocidad inicial = Vi (m/s)
  • velocidad final = Vf (m/s)
  • velocidad promedio = Vprom => Vx (m/s)
    • Distanciadistance
  • distancia inicial = Xi (m)
  • distancia final = Xf (m)
  • distancia promedio= Xprom => X (m)
    • Tiemposchedule
  • tiempo = t (seg)
Ejercicios
    velocidad promedio
  • Vprom = ΔX/Δt
    • distancia final
  • Xf = Xi+Vprom.t

Movimiento Parabólico

Trayectoria en un campo bi dimensional

El movimiento parabólico es el tipo de movimiento que sigue un objeto cuando se lanza al aire con una velocidad inicial que forma un ángulo con la horizontal.

Sus magnitudes son las siguientes:

    Aceleraciónbolt
  • aceleracion = a (m/s2)
    • Velocidadspeed
  • velocidad inicial = Vi (m/s)
  • velocidad final = Vf (m/s)
  • velocidad promedio = Vprom => Vx (m/s)
    • Distanciadistance
  • distancia inicial = Xi (m)
  • distancia final = Xf (m)
  • distancia promedio= Xprom => X (m)
    • Tiemposchedule
  • tiempo = t
Ejercicios
    aceleracion
  • a = ΔVprom/Δt
    • velocidad final
  • Vf = Vi+a.t
  • Vf2 = Vi2+2a.(Xf-Xi)
    • distancia final
  • Xf = Xi+1/2(Vi-Vf).t
  • Xf = Xi+Vi.t+1/2a.t2
Aclaraciones:
  • m: Metros (unidad de medida de distancia)
  • s/seg: Segundos (unidad de medida de tiempo)
  • X: Representacion de distancia horizontal
  • Y: Representacion de distancia vertical
  • "x"2: Exponente al cuadrado
  • gravedad: Su valor es constante (9.8 m/s2 )
  • m/s: Metros sobre segundos (Unidad de medida de velocidad)
  • m/s2: Metros sobre segundos al cuadrado (Unidad de la aceleracion)

Las unidades de medidas dadas en esta presentacion se utilizan de manera standar, lo que significa que se pueden realizar o conversionar a otros tipos de unidades a conveniencia del ejercicio.
